# 🇬🇧 SpeakBritish

A clean, split-panel web app for practising **UK English pronunciation**. Type or paste any word, sentence, or essay — and hear it read aloud in a proper British (en-GB) voice.

## ✨ Features

- **Real-time word highlighting** — each word lights up as it's spoken
- **UK English voices** — automatically selects en-GB voices from your browser
- **Speed & pitch control** — slow it down to hear sounds clearly
- **Quick phrases** — 8 pre-loaded British phrases including tricky city names
- **Animated waveform** — visual feedback while audio plays
- **Pronunciation tips** — explains key UK vs US English differences

## 🚀 Usage

Just open `SpeakBritish.html` in any modern browser — no install needed. It's a single self-contained HTML file.

Or run the full React source:

```bash
pnpm install
pnpm dev
```

## 🗣️ Best Voices

Look for these in the voice dropdown for the most authentic UK accent:
- **Daniel** (Male, British)
- **Serena** / **Hazel** (Female, British)

## 🧠 UK Pronunciation Tips

| Feature | UK English |
|---|---|
| Non-rhotic | 'car' → 'cah' (R silent after vowels) |
| Long A vowel | bath, grass, dance → long 'ah' sound |
| Crisp T | 'butter' stays 'butter', not 'budder' |
| Tricky cities | Leicester = Lester, Worcester = Wooster |
